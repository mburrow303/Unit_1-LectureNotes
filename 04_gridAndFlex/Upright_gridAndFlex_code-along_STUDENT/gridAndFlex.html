Grid Heading

Buy Nuts!
Grid is a 2-Dimensional layout system. It can layout entire sites or smaller components. It allows for precise item placement within containers 
display: grid;
 sets a grid for the containter used such as divs, main, sidebar, and many others. These contain elements on the page, determines how they are laid out inside the container. Only affects direct child elements. It is governed by columns and rows.

Flex Heading

Accomplish

Horizontal or Vertical alignment of elements

Arrangement of elements

Space distribution of elements

Quick Notes

It's one-dimensional. Either row or column, but not both at the same time

Useful for layout on nav bars, headers, image galleries

Place elements that are direct children, but does not affect further children

Set how a flex item with shrink or growwill fit in the space of the container

Flex Container

Applies to the container andit's children

Must be defined in order to be applied

Justify Content

Align Items

Similar to Jusify Content but works along the cross-axis (perpendicular to the flex-direction)

  3. center

Centering Content

Uses center from both justify-content andalign-items
